# Getting started

### Make sure you have docker installed and running (https://www.docker.com/products/docker-desktop/)

1. Clone the repository:
    ```
    git clone https://github.com/Tobibit/chess.git

2. Navigate to the project directory
    ```
    cd chess

3. Build the Docker containers
    ```
    docker compose build

4. Start the application
    ```
    docker compose up

5. Open your browser and go to http://localhost:8001

<br>

6. To stop the application, use 'Ctrl+C' in the terminal, and then run:
    ```
    docker compose down
